# TwitchFighter

Feel-Good online gambling site for everything related to Street Fighter fights

## Getting Started

```
npm -v
npm install -g npm@6.12.0
npm install -g yarn@$(cat .yarn-version)
```

## How to build

```
nvm install 10.13.0
nvm use
yarn
```

## How to run

```
yarn compile
node dist/hello_world.js
```

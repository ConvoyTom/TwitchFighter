# TwitchFighter

Feel-Good online gambling site for everything related to Street Fighter fights

## Getting Started

Install node.js
Install homebrew
Install npm

```
npm -v
npm install -g npm@6.13.4
npm install -g yarn@$(cat .yarn-version)
```

## How to build

```
nvm install 13.6.0
nvm use
yarn
```

## How to run

```
yarn compile
yarn start
```

point your browser to localhost:3000
